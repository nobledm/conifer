Conifer UI

What is Conifer UI?

The word Conifer was a simple theme chosen at complete random with the main purpose of being a backdrop design for a string of assignments. However, what fun would it be to just do the bare minimum? Thus it has become an exercise in basic best practices and simple css tricks built from scratch with Sass.

This particular version is the very first! The UI part of its name comes from the library that was first on the list: JQuery UI.

Enjoy!
